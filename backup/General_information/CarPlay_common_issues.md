# CarPlay common issues

Things to check if you have issues with CarPlay connection.

## Vehicle side

* Make sure your main unit supports CarPlay.
* CarPlay FEC/SWaP code is installed and valid (enter [hidden menu](/doc/key-combinations-and-shortcuts-7tk8NfVoLo) and check [FEC list](/doc/fec-list-for-vag-mib2-mib3-cL1qwRAePx)).
* CarPlay functionality is correctly enabled in `5F` adaptations.
* Car is equipped with proper USB port ([video](https://youtu.be/7yTYaxzeXfY)).
* USB is connected with 4+2 wires.
* USB port is coded correctly.

## iPhone side
