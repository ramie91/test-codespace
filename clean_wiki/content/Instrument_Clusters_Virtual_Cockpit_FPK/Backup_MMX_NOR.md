# Backup MMX NOR


:::warning
Is important to have a backup if you start playing with MMX

:::

Connect to UART with TTL cable and connect USB pen drive to USB port as described [here](https://mibwiki.one/doc/ttl-and-usb-connection-BwIQFfVDTp)

Login via TTL [as root](https://mibwiki.one/doc/mmx-root-passwords-Ksdm2xg8md) and enter following command (will take about 1-5 minutes):


:::tip
cat /dev/fs0 > /fs/usb0/mmx_fs0

:::
