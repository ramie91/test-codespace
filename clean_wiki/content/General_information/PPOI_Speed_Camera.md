# PPOI Speed Camera


:::info
The Speed cameras are classified as either ‘Fixed‘ or ‘Mobile‘ (as you know, only up to 10 categories can be sellected)

:::


:::tip
MST2, MHI2 non-Audi, MHI2 Audi are using three separate packages. Make sure you download the correct one.

:::

## Download

* Download latest PPOI packages from [this telegram group](https://t.me/mibspeedcam).
* Enter unit SW train into [mib-helper.com](http://mib-helper.com)  and get links to available PPOI packages.

## Installation instructions

### Audi MMI Navigation Plus (MHI2)


1. Take an SD card, format it with `FAT32` file system, with smallest allocation size available.
2. Download PPOI update package and unzip it.
3. Copy the unpacked PPOI package to the root directory of the SD card.
4. Turn the MIB unit on and wait about 1 minute for it to fully start.
5. Place the SD card in `SD1` port of the main unit.
6. Go to `MENU` →  `Setup MMI` →  `System Update` →  `SD1`→ `Start Update`.
7. When update is completed press `OK`.
8. Go to `NAV` → `Settings` → `Map content` and confirm that new PPOIs are enabled.
9. Go back to the `NAV` and check if PPOIs are visible on the map.

### VW / Seat / Skoda


 1. Take an SD card, format it with `FAT32` file system, with smallest allocation size available.
 2. Download PPOI update package and unzip it.
 3. Copy the unpacked PPOI package to the root directory of the SD card.
 4. Turn the MIB unit on and wait about 1 minute for it to fully start.
 5. Place the SD card in `SD1` port of the main unit.
 6. Go to `NAV` →  `Setup` →  `Manage Memory` →  `Update My POIs` → `Update` → `Next`
 7. When update is completed press `OK`.
 8. Go to `NAV` → `Setup` → `Map` and enable `Show POIs` option.
 9. Go to `NAV` → `Setup` → `Map` → `Select categories for POIs`, scroll to the bottom of the list, and enable new categories.
10. Go to `NAV` → `Setup` → `Nav. announcements` and enable option to note personal POIs.
11. Go back to the `NAV` and check if PPOIs are visible on the map.
