# USB extensions (ports and HUBs)

\
## CarPlay


:::info
As far as functionality goes, CarPlay requires [specific hardware](/doc/usb-port-carplay-android-auto-GOvU5Ji0LN) to work with MIB. Both Android Auto and MirrorLink will work with [any USB ports](/doc/usb-port-d-link-android-auto-only-aGZ7ofACzq).

:::

### `5Q0 035 726`

### `5Q0 035 726 B` USB port - (short - white connector)

* 0,5A current

### `5Q0 035 726 C` USB port - (long - black connector)

* 0,5A current

### `5G0 035 953 D` USB HUB

### `5G0 035 222 E` assembly (`5Q0 035 726 E` USB port + AUX port - long - green connector)

\[& `6C0 035 249 B`\]

* AUX port
* USB iOS (`UI4`) support
* Standard Infotainment Unit (`I8E`) support
* High Infotainment Unit (`I8H`) support
* Android/MirrorLink support
* 1,5A current

### `5G0 035 222 F` Assembly (`5Q0 035 726 F` USB port + AUX port)

* AUX port
* limited USB (`UE4`) support
* Standard radio Gen2 (`I8E`) support
* High Infotainment Unit (`I8H`) support
* Android/MirrorLink support

### `5Q0 035 726 G` USB port (purple connector) \[& `6C0 035 726 C`\]

* limited USB (`UE4`) support
* Standard radio Gen2 (`I8E`) support
* Android/MirrorLink support
* 0,5A current

### `5G0 035 222 H` Assembly (5Q0 035 726 J USB port + AUX)

* AUX support
* USB iOS (`UI4`) support
* Standard radio Gen2 (`I8E`) support
* Android/MirrorLink support

### `5Q0 035 726 J` USB port - (long - black connector)

* USB iOS (`UI4`) support
* Standard Infotainment Unit (`I8E`) support
* High Infotainment Unit (`I8H`) support
* Android/MirrorLink support

### `5Q0 035 726 M` USB port - (long - black connector)

* If used with MIB2 STD, only Android Auto and iPhone charging works (CarPlay does not work)
