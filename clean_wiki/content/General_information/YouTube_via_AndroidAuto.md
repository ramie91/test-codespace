# YouTube via AndroidAuto


:::info
For youtube on android auto you can use AAAD app

:::


1. Install AAAD from github: <https://github.com/shmykelsa/AAAD/>
2. Download `CarStream` or `AAmirror`.


:::info
If you use AndroidAuto in vehicle with touchscreen download `CarStream`.

If you use AndroidAuto in vehicle without touchscreen use `AAmirror`.

:::


3. Accept all permissions and that is it.
