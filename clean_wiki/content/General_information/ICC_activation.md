# ICC activation

ICC, in-car communication, voice enhancer, intercom… Basically rear speakers will play live audio feed from front microphone. Designed to improve communication between driver and passengers of large (or loud) vehicles.

## Requirements


1. MIB2 unit that supports `00070400` FEC/SWaP code and is equipped with front microphone,
2. Diagnostic tool capable of parameterization / sending dataset (VCP, OBDeleven, CarScanner).

## Procedure


1. Press and hold `MENU` for 10 seconds, go to red engineering menu, and check if code `00070400` is supported by unit.
2. If code is not supported, you will have to upgrade firmware to more recent one.
3. Check if the code `00070400` is listed as installed and valid.

   If your unit is patched, you can self-generate the code and upload it to the unit. EL-patch should also enable this feature.
4. Use VCP, OBDeleven, or CarScanner to upload ICC dataset to module.
