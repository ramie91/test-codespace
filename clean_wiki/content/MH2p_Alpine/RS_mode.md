# RS mode

In byte 2 of 5F coding, set bits like follows:

bit 4=0\nbit 5=1\nbit 6=1\nbit 7=0 


:::warning
**IMPORTANT!** [Reboot](https://mibwiki.one/doc/key-press-combos-5nh4BeCC9S) the unit and it should show RS monitor in CAR menu:

:::
